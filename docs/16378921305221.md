# Java 与 Leetcode 总结

1、队列、栈、链表 都可以通过LinkedList实现

2、其实很少使用ArrayList、大部分使用了LinkedList

3、LinkedList 是一个继承自 AbstractSequentialList 的双向链表，因此它也可以被当作堆栈、队列或双端队列进行操作

4、在二叉树的层序遍历中使用到了ArrayList

5、ArrayList 实现了 List 接口，继承了 AbstractList 抽象类，底层是基于数组实现的，并且实现了动态扩容

有如下实现关系：

- Collection-->Queue-->Deque-->LinkedList(实现类)

- 对于队列来说有offer、poll 、peek 操作

- 对于stack来说有push 、pop 操作

- 对于ArrayList来说有 add、remove操作

- LinkedList几乎实现了所有接口，offer、poll 、peek、push 、pop、add、remove

- 对于双端队列

- size() 函数可以获取大小

- 对于for循环，和OC一样 for (int i=0;i<10;i++){}
