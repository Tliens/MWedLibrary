# 网络组件

#### 1.底层基于Alamofire封装

1. 利用OperationQueue、Operation控制并发量、优先级（Alamofire的并发控制会影响timeout计时）
2. 重试机制，字典线程锁
3. 支持每个接口不同的timeout设置
4. 支持接口缓存
5. 支持IP直连，IP列表
6. 支持自定义错误码处理回调
7. 支持http 验证赦免名单
8. 支持网络状态监听回调（未连接网络停止重试机制）
9. 支持接口冷区期
10. 线上达到0崩溃

#### 2.业务层 rx + 原生
1. 每一个网络请求的结果最终都会包装成Observable<[String: Any]> 
2. Observable的生命周期由调用者确定
3. 针对OC，则采用用block回调形式
