# Funlink工作日志

1. iphone5 崩溃
```Swift
let res = Int(arc4random())%20
```
崩溃原因：arc4random()返回 UInt32, 在32位处理器中，Int 默认为 Int32，Int 是从 [-2^30+1,2^30-1],UInt 是从 [0,2^31-1] ,Int类型强转UInt32，可能会导致崩溃。在64位操作系统中则没事，应为Int默认为Int64

2. 模拟器运行app 无法运行成功
原因：i386、x86_64、armv7、armv7e，其中i386、x86_64与模拟器相关，有些第三方pod不支持i386、x86_64，此外自 Xcode 12.3 起，Apple 不允许在一个 .framework 文件中绑定多平台的库，需要使用.xcframework
